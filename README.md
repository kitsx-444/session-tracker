# session-tracker

### What is this program:
This program is a vital tool for all traders that want to align themselves with the open times of major trading sessions. 

### What it does:
It receives the user offset input and lets them know the currently active session and upcoming event within the week.

1. How to output the current time.
2. Deeper familiarity with datetime modules, strptime and strftime.
